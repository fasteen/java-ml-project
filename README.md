# 🚀 Java Machine Learning Project (MiniGPT Style)

This project demonstrates building Machine Learning models from scratch using pure Java — without relying on external ML frameworks.

## 🔥 Features

- ✅ Linear Regression (from scratch)
- ✅ Custom Neural Network logic
- ✅ Forward & Backward Propagation implemented manually
- ✅ Next Word Prediction Model (MiniGPT-style)
- ✅ Tokenization and vectorization logic
- ✅ Training loop with loss calculation
- ✅ Console-based prediction using Scanner

---

## 🧠 What Makes This Special?

Unlike typical ML projects using Python libraries, this project:

- Uses **pure Java**
- Implements **core ML concepts manually**
- Helps understand **how models actually work internally**
- Acts as a **foundation for building LLMs (like GPT)**

---

## ⚙️ Tech Stack

- Java
- Core Java Libraries (No TensorFlow / PyTorch)
- Custom ML logic
